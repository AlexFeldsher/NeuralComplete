  • Description

    This extension lets you train an LSTM neural network by automatically capturing your input in text areas, and the trained network is used to provide you with autocomplete suggestions.
    The network is trained with strings of length that is defined in the 'Neural Network Configuration' tab.
    Changing the structure of the neural network by modifying the hidden layers will reset the network and it will require you to retrain it.

    Addtionally you can define "shortcuts" that will overide the neural network's suggestions.

    User input isn't saved for future training.

    Usage

    Press Ctrl-Space to apply autocompletion.
